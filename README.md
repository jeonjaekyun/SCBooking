# SCBooking

## 주제
Spring을 활용한 스터디 카페 예약 프로그램

## 개발 언어 및 IT Skill
- JAVA, Javascript, JSP
- Spring Framework, STS
- MySQL, MyBatis
- Apache Tomcat

## 주요기능
- 로그인/회원가입
- 회의실 예약
- Q&A 게시판을 통한 관리자와 커뮤니케이션
- 마이페이지(개인정보 관리, 예약내역, Q&A 리스트)
